# 🚀 TongueCare 舌診系統 - 本機端啟動指南

學長好！這套舌診系統分為「前端網頁介面」與「後端 AI 大腦」。
為了確保系統能在你的電腦上獨立、完整地運行，請在打開網頁前，先完成以下三個啟動步驟：

### 🛠️ 步驟 1：確認前端連線網址 (app.js)
請打開 `app.js`，找到大約第 130 行的 `API_URL`。
確認網址已經設定為本機端，這樣網頁才找得到你的本機伺服器：
> const API_URL = 'http://127.0.0.1:8080/api/analyze';

### 🔑 步驟 2：設定老中醫的 API 金鑰 (.env)
確認資料夾(/Users/.../Tongue-Diagnosis-main/.env)裡有 .env 檔案
系統需要呼叫 Google Gemini 進行深度推論。請確認專案資料夾內有一個 `.env` 檔案。
請在裡面放入 API 金鑰（注意：這個檔案包含私密金鑰，請勿上傳至公開的 GitHub）：
> GEMINI_API_KEY=你的_API_金鑰_請貼在這裡

### 🧠 步驟 3：喚醒 AI 大腦 (啟動 FastAPI 後端)
請在 VS Code 打開終端機 (Terminal)，確認環境已安裝好所需的套件後，輸入以下指令啟動伺服器：
> uv run python src/api_server-up.py

(如果終端機顯示 `✅ ONNX 大腦初始化成功` 以及 `Uvicorn running on http://127.0.0.1:8080`，代表大腦已順利上線！)

---
### 🎉 步驟 4：開始看診！
大腦啟動後，請不要關閉終端機。直接在資料夾中點擊兩下打開 `main.html`，就可以開始體驗完整的 AI 舌診、漸進式揭露以及電子病歷匯出功能了！



前端網頁啟動建議：
為了避免瀏覽器阻擋相機權限，建議學長解壓縮後，在html資料夾內打開終端機輸入：
python3 -m http.server 5500
然後在瀏覽器打開 http://localhost:5500 來獲得最完整的操作體驗喔！點這個連結
